# House Price Prediction System

A Machine Learning project that predicts house prices using Linear Regression based on features like area, bedrooms, bathrooms, stories, and parking spaces.

---

# Features

- House price prediction using Machine Learning
- Data preprocessing
- Linear Regression model
- Model evaluation
- Data visualization
- Custom price prediction
- Graphical analysis using Matplotlib

---

# Technologies Used

## Programming Language
- Python

## Libraries
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
